# MyTown
我的小镇，任何人可以建立自己的小镇，例如某个游戏为主题的小镇，或者有公共目的的Social小镇，未来可以链接在一起；也可以共建一个City：由无数个小镇和小区组成。



## Dev & Resource
想基于已有的开源的repository去开发一个桌面应用，就桌面应用（全部功能）+Web应用（部分功能受限制）+浏览器插件（部分功能受限制）。
因为是一个交互游戏，想基于Godot开发。
我们使用付费的一个Godot插件：https://gdaimcp.com/docs，来开发Godot应用。
